# **App Name**: DroidForge AI

## Core Features:

- AI Project Generation: Takes in the user prompt and interprets it to correctly output the correct file structures, code content, and dependencies that are to be incorporated into the files. Uses LLM reasoning to decide which elements or tools to incorporate, as opposed to literally outputting the information requested.
- Code Preview Panel: Provides a tabbed interface for viewing code, split into Java/Kotlin, XML, Gradle, and Manifest sections. All code previews include syntax highlighting.
- Download Manager: Enables downloading the entire generated project as a ZIP file, and downloads for each file.
- File System Generator: Generates a standard Android project directory structure, including source, resource, and Gradle files.
- Project Structure Display: Displays a preview of the generated project file system in a collapsible tree format. Enables the user to quickly navigate all files in the project.

## Style Guidelines:

- Primary color: Medium green (#4CAF50), reminiscent of Android green.
- Background color: Very light gray (#F0F0F0).
- Accent color: A deep, dark, slightly cyan-tinged teal color (#008080).
- Body and headline font: 'Inter', a sans-serif for a clean and modern interface.
- Code font: 'Source Code Pro' for the display of generated source code.
- Use clear divisions between sections to help users easily understand relationships and functions. The layout shall use common Android paradigms so as not to confuse Android app developers.
